# SentInvest 项目启动说明书

本文档用于快速启动本项目的前端、后端、数据爬虫和自动分析调度脚本。

## 1. 项目结构速览

- `frontend/`：Vue3 + Vite 前端
- `backend/`：FastAPI 后端 API
- `scraper/`：东方财富股吧数据爬虫
- `pipeline/`：Spark 清洗 + FinBERT 分析 + 风险画像
- `run_pipeline.py`：自动调度脚本（每 3 小时跑一轮分析）

---

## 2. 环境准备

建议：

- Python 3.10+
- Node.js 18+
- MySQL 8+
- （推荐）Java 8/11（PySpark 运行需要）

### 2.1 安装 Python 依赖（项目根目录执行）

```bash
pip install -r requirements.txt
pip install -r backend/requirements.txt
pip install -r scraper/requirements.txt
pip install -r pipeline/requirements.txt
pip install schedule
```

### 2.2 安装前端依赖

```bash
cd frontend
npm install
```

### 2.3 配置环境变量

项目通过根目录 `.env` 读取数据库连接，至少需要：

```env
DATABASE_URL=mysql+aiomysql://用户名:密码@主机:端口/数据库名
```

---

## 3. 数据库初始化（首次必做）

### 3.1 建表

在项目根目录执行：

```bash
python backend/app/core/init_db.py
```

### 3.2 初始化理财产品数据（推荐）

```bash
python backend/scripts/seed_products.py
```

---

## 4. 后端启动方式

后端默认端口：`8000`

```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

健康检查：打开 `http://127.0.0.1:8000/`，应返回：

```json
{"msg":"SentInvest backend is running"}
```

---

## 5. 前端启动方式

前端默认端口：`5173`

```bash
cd frontend
npm run dev
```

浏览器访问：`http://localhost:5173/`

> 前端 API 默认请求 `http://127.0.0.1:8000`，请确保后端已启动。

---

## 6. 数据爬虫启动方式

爬虫入口：`scraper/main.py`

### 6.1 常用启动命令

在项目根目录执行：

1) 冷启动 + 实时轮询（推荐生产运行）

```bash
python scraper/main.py
```

2) 只跑一次冷启动后退出

```bash
python scraper/main.py --once
```

3) 跳过冷启动，直接实时轮询

```bash
python scraper/main.py --poll-only
```

4) 自定义冷启动页码范围（历史回填）

```bash
python scraper/main.py --start-page 1 --end-page 100 --once
```

日志文件：`logs/scraper.log`

---

## 7. 自动分析脚本启动方式（调度引擎）

自动分析入口：`run_pipeline.py`

```bash
python run_pipeline.py
```

调度逻辑：

- 启动后立即执行一轮：
  1. `pipeline/cleaner.py`（Spark 清洗）
  2. `pipeline/analyzer.py`（FinBERT 情感分析）
  3. `pipeline/profiler.py`（用户风控画像）
- 之后每 **3 小时**自动执行一轮
- 可按 `Ctrl + C` 安全退出

---

## 8. 推荐启动顺序（全链路）

建议开 4 个终端，按顺序执行：

1. **爬虫终端**

```bash
python scraper/main.py
```

2. **自动分析终端**

```bash
python run_pipeline.py
```

3. **后端终端**

```bash
cd backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

4. **前端终端**

```bash
cd frontend
npm run dev
```

## 9. 常见问题排查

### 9.1 后端启动失败（数据库连接错误）

- 检查根目录 `.env` 的 `DATABASE_URL`
- 确认 MySQL 已启动、库名已创建、账号有权限

### 9.2 前端页面无数据

- 确认后端 `8000` 端口可访问
- 检查浏览器控制台和后端终端日志

### 9.3 pipeline 报错（Spark / Java）

- 确认已安装 Java（并配置 `JAVA_HOME`）
- 确认 `pipeline/requirements.txt` 依赖已安装

### 9.4 自动调度未触发

- `run_pipeline.py` 需要持续运行，不要关闭终端
- 首次启动会先立即执行一轮，后续每 3 小时触发
